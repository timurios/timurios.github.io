Engineering Projects

ECE3337 Electromagnetics Project Lab (Spring 2025): Design and Implementation of Semiconductor Photoconductive Switch for High Frequency Applications

This project focuses on the design and characterization of an optically controlled photoconductive microwave switch operating in the 1–4 GHz range (L and S bands). The switch is activated by High-Power LEDs (≤ 1 W optical power each) and is designed to meet strict performance criteria: insertion loss within ≤1 dB in the closed state and isolation ≥10-20 dB in the open state. The final design is enclosed in shielded packaging for practical use and controlled via a designed external TTL gate driver for seamless integration into electronic systems.

ECE 3332 Microcontrollers Project Lab (Fall 2025): Time’s Up - Implementation of long-range RFID warehouse management system

This project explores the design and implementation of a portable warehouse management system called “Time’s Up”, developed to identify expired products from a distance without requiring direct manual intervention. The system integrates high-power Ultra High Frequency (UHF) RFID technology based on the ThingMagic M6E reader module, configured to operate at 27–30 dBm power levels for long-range identification.

ECE 3331 FPGA Robotics Project Lab (Fall 2024): Design and Implementation of an Autonomous Office Delivery Robot Using FPGA-Based Control Systems

This report presents the design and implementation of an office delivery robot, developed using a Basys3 FPGA board, as part of the ECE 3331-003 Robotics Project Lab coursework. The robot integrates multiple subsystems to perform efficient package delivery tasks. Key features include real-time package color detection using a TCS3200 color sensor, enabling accurate identification of items; infrared (IR) communication with a counterpart operator via a linked LED array, facilitating seamless task updates; and overcurrent protection to ensure safe operation of the motors.

3D Printed Optical Equipment

I designed and implemented a custom optical system using 3D-printed mounts and housings to prototype and align a quantitative phase imaging setup rapidly. This approach enabled flexible iteration of optical geometries, reduced cost, and improved mechanical stability while maintaining precise beam alignment, demonstrating the utility of additive manufacturing for experimental photonics.
